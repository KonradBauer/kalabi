Collections (files under src/collections/)

   (client/location/year/scope/materials), meta, order
4. ProjectCategories.ts - name, slug, description, order

6. Testimonials.ts - author, role, content, rating, project (rel), featured, order
7. ContactSubmissions.ts - name, email, phone, subject, message, read checkbox; public create access

Globals (files under src/globals/)

1. Header.ts - logo, navItems array (label/link/newTab), ctaButton group
2. Footer.ts - logo, description, columns array (title + links), contactInfo, socialLinks, copyrightText
3. SiteSettings.ts - siteName, siteUrl, logo, favicon, ogImage, companyInfo group, seo group (defaultTitle/titleTemplate/defaultDescription)
4. HomePage.ts - hero, aboutPreview (with stats array), servicesSection, projectsSection, ctaBanner, testimonialsSection, meta
